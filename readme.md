## Funcionalidades:
Inicia contagem em ordem progressiva.
Pausa a contagem deixando a cor dos números vermelha.
Zera a contagem.

## Tecnologia usada:
HTML5, Javascript e CSS3.

## Aprendizados:
Manipulação de setInterval e cleanInterval.
DOM e eventos.
Lógica de programação.
